# Anty-Detect Browser

Локальний анти-детек браузер для керування ізольованими браузерними профілями.

## Опис

Десктопний додаток для керування ізольованими браузерними профілями. Кожен профіль має власні кукі, кеш, історію та налаштування проксі, що дозволяє працювати з декількома акаунтами одночасно без ризику їхнього взаємозв'язку.

## Технологічний стек

- **Python 3.10+**
- **Flet** - сучасний GUI на базі Flutter
- **Playwright** - керування браузерними профілями (Chromium)
- **SQLite** - зберігання метаданих профілів та проксі

## Встановлення

1. Клонуйте репозиторій або завантажте файли проєкту.

2. Встановіть залежності:
```bash
pip install -r requirements.txt
```

3. Встановіть браузери для Playwright:
```bash
playwright install chromium
```

## Запуск

```bash
python main.py
```

## Функціональність

### Керування профілями
- Створення профілів з унікальним ID
- Автоматичне створення папок профілів
- Збереження сесій (кукі, паролі, вкладки)
- Запуск та зупинка профілів
- Редагування та видалення профілів

### Проксі
- Налаштування проксі для кожного профілю
- Підтримка HTTP, HTTPS та SOCKS5
- Автентифікація (логін/пароль)
- Менеджер збережених проксі

### Інтерфейс
- Сучасний UI з підтримкою тем (Dark/Light)
- Бокове меню навігації
- Таблиця профілів зі статусами
- Діалоги створення та редагування

## Структура проєкту

```
│── main.py            # Точка входу (Flet UI)
│── browser_logic.py   # Робота з Playwright
│── database.py        # Робота з SQLite
│── profiles/          # Папка з даними браузерів
│   ├── {profile_id}/  # User Data Dir профілю
│   └── ...
│── browser_profiles.db # База даних SQLite
└── requirements.txt   # Залежності
```

## Використання

1. **Створення профілю:**
   - Натисніть "Новий профіль" або кнопку "Створити профіль"
   - Введіть назву профілю
   - Додайте нотатки та теги (опціонально)
   - Виберіть проксі (опціонально)

2. **Запуск профілю:**
   - Натисніть кнопку ▶️ (Play) біля профілю
   - Браузер відкриється з ізольованим профілем

3. **Налаштування проксі:**
   - Перейдіть до розділу "Проксі"
   - Додайте новий проксі з необхідними параметрами
   - Призначте проксі профілю при створенні або редагуванні

4. **Теми:**
   - Перемикайте тему за допомогою перемикача в правому верхньому куті

## Примітки

- Кожен профіль зберігає свої дані в окремій папці `profiles/{profile_id}/`
- Дані профілю (кукі, паролі, історія) зберігаються автоматично
- При видаленні профілю папка з даними також видаляється
- Переконайтеся, що Chrome встановлений для використання `channel: "chrome"`
